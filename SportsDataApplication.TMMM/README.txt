                         --------DESCRIPTION--------
This application provides users with access to a wide range of sports statistics, including player stats, team standings, match results, and upcoming fixtures.


			*********Key Features*********

			     User Authentication: 
Secure login system that allows users to create accounts and sign in securely.

			     Sports Data Integration: 
Databases to fetch sports data for various sports leagues and events.

			     Dashboard: 
User-friendly dashboard where users can easily navigate and access different sports and leagues.

			     Player and Team Stats: 
Detailed statistics for players and teams, including performance metrics, rankings, and historical data.

			     Match Information: 
Display match results, upcoming fixtures, and schedules.

			     Search and Filter:
Advanced search and filter functionalities to help users find specific teams, players, or matches quickly. 

			     Customization: 
Personalized experience by selecting favorite teams, customizing notifications, and setting preferences.


		    ********* System Requirements *********

			- IDE: Visual Studio 2026
		- Framework: .NET 6.0 / .NET Framework 4.8
	      - Database: SQL Server 2022 (v16.x) or higher 
     * Note: The .mdf file requires a LocalDB or SQL Server v17 instance.


                    ********* How To Run the App *********

	1. Open the 'SportsStatsApp.sln' file in Visual Studio 2026.
	2. Build the solution (Ctrl+B).
	3. Press F5 to Run.
	*Note: Ensure the database file is in the Debug/bin folder.
